# LlamaBoss Beta v0.1.0

LlamaBoss Beta v0.1.0 marks the shift from a local chat client into a local desktop AI assistant with projects, skills, notes, files, and approval-gated tools.

## Highlights

- Repositioned LlamaBoss as a private local desktop AI assistant for Windows
- Added project support with long-lived project folders
- Added project source folders, templates, notes, outputs, and workflows
- Added global Skills for reusable cross-project workflows
- Added project-aware context loading through `PROJECT.md`
- Added project notes and global notes integration
- Added per-chat workflow folders for imported and generated artifacts
- Added approval-gated Python script creation and execution
- Added helper tooling for CSV, XLSX, PDF, DOCX, Python, notes, and file operations
- Improved controlled tool routing and agent behavior
- Improved cleanup behavior for chat-associated workflow files
- Updated product direction around local LLMs, files, projects, skills, and approved tools

## Beta notice

This is beta software. Features, workflows, file handling, installer behavior, and UI details may change before a stable release.

Use caution with important documents and test beta workflows on copies first.

## Known areas still being polished

- Model download UX and error messages
