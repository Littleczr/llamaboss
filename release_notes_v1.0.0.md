LlamaBoss v1.0.0 — Lightweight Edition
A lightweight native desktop chat client for Ollama, built with C++ (wxWidgets + Poco).

Features:
- Streaming chat with any Ollama model
- Group chat — invite two models into one conversation (round-robin or @directed)
- Model picker (click pill to switch, right-click to manage)
- Model Manager — pull, delete, and create custom models (Modelfile Creator)
- Streaming markdown renderer (headings, bold/italic, code blocks with copy)
- Dark & light themes with live switching
- Image attachment (file, drag-drop, Ctrl+V clipboard paste)
- Text file attachment (21 extensions, 100KB cap)
- Conversation sidebar with auto-save, multi-select, and delete
- Conversation persistence (JSON) with Ctrl+S/O/N shortcuts
- Resizable sidebar with width persistence
- Window state persistence (position, size)
- Think-block support (collapsible <think> rendering)
- Fast drag-scroll in chat display

Requirements:
- Windows 10/11 (x64)
- Ollama running locally
